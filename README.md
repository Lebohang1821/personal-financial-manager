```bash
## Installation

To get started with Personal Financial Manager, follow these steps:

1. Clone the repository to your local machine:

```
git clone https://github.com/Lebohang1821/personal-financial-manager.git

    Navigate to the project directory:


cd personal-financial-manager

    Install dependencies:

npm install

    Start the development server:

npm start

    Open your browser and navigate to http://localhost:3000 to view the application.

Usage

Once the application is running, users can:

    Track their expenses by adding transactions.
    Set budgets for different expense categories.
    View graphical representations of their spending habits.
    Analyze their financial data to make informed decisions.
